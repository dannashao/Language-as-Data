This is the code for Lab 4 of the course Language as Data 2023, Vrije Universiteit Amsterdam.
It should only be used for the purpose of the course and not be re-distributed. 

Author: Lisa Beinborn


In this lab, we work on the analysis of terminology. Start with lab 4.1 in the folder code.

You need to install the following packages: 
numpy, sklearn, wordcloud, gensim, networkx

Download the English fasttext vectors wiki-news-300d-1M.vec from https://fasttext.cc/docs/en/english-vectors.html and put the file into the data folder. 

You will also need an access key from Babelnet: https://babelnet.org/register
