This is the code for Lab 3 of the course Language as Data 2021, Vrije Universiteit Amsterdam. 
It should only be used for the purpose of the course and not be re-distributed. 

Author: Lisa Beinborn


In this lab, we see examples for analyzing the syntactic structure of sentences and for extracting stylistic features. We also focus on different plotting options. 

Download the file mrc.dct from https://github.com/samzhang111/mrc-psycholinguistics/blob/master/mrc2.dct and save it in your data folder.
